Store Rating Management System

A full-stack web application that allows users to register, log in, explore stores, and submit ratings between 1 to 5. 
The platform supports role-based access for System Administrators, Normal Users, and Store Owners.

🚀 Tech Stack
Frontend: React.js
Backend: Express.js
Database: PostgreSQL / MySQL
Authentication: JWT & bcrypt

✨ Features
User Authentication & Authorization
Role-Based Access Control
Store Listing & Search
Submit and Update Ratings
Admin Dashboard
Store Owner Dashboard
Sorting & Filtering
Responsive UI Design
Secure Password Validation

👥 User Roles
System Administrator
Manage users and stores
View platform statistics
Apply filters and sorting
Normal User
Register/Login
Search stores
Submit & modify ratings
Store Owner
View store ratings
Check average rating

✅ Validations
Name: 20–60 characters
Address: Max 400 characters
Password: 8–16 characters with uppercase & special character
Valid email format required
